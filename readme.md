Description:
This project automates complex financial logic for mortgage management and analytics. It integrates multiple data sources to provide real-time insights and decision-making support.

Tech Stack:

Tool: n8n (Self-hosted/Cloud)

Logic: Complex conditional branching, data transformation.

Integrations: Bank APIs, LLMs

Key Features:

Automated mortgage calculation based on variable inputs.

Financial data validation and integrity checks (QA-driven approach).

Notification system for critical financial milestones.

How to use:

Install n8n.

Import the .json file from the workflows/ folder.

Configure your own credentials for the connected services.
